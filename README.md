## Compile scss automatically
### 1. npm init
### 2. npm install 
### 3. npm install node-sass
### 4. Create a file naming package.json have content as bellow
 `"scripts": {
    "scss": "node-sass --watch scss -o css"
  }`


### 4. npm run scss
